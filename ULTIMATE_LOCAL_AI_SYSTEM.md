## ARCHITECTURE LAYERS

### Layer 1: Base LLMs
- **Llama 2 7B** (fine-tuned with LoRA)
- **Mistral 7B** (for speed, streaming, voice)
- **Phi-3 3.8B** (fallback for low-power devices)

### Layer 2: Specialized Classifiers
- **Category Classifier**: FastText + SVM
- **Merchant Normalizer**: Fuzzy matching + ML

### Layer 3: Predictive Models
- **LSTM Mood Predictor**: 7-day lookback window
- **Focus Duration Predictor**: Neural net attention mechanism
- **Goal Success Forecaster**: Logistic regression
- **Spending Anomaly Detector**: Isolation Forest (unsupervised)

### Layer 4: NLP & Speech
- **NER (Named Entity Recognition)**: Extracts merchants, dates, amounts
- **Sentiment Analysis**: RoBERTa fine-tuned on journal entries
- **Speech-to-Text**: Whisper.cpp (runs on CPU, 95% accuracy)

### Layer 5: Ensemble & Meta-Learning
- **Voting Ensemble**: LLM + Regex + NER weighted by confidence
- **Stacking Ensemble**: Base models -> Meta-learner
- **Adaptive Learning Loop**: Retrains models weekly from user corrections
